Inputs & Combining Gates

Real game objects, timing, and building circuits with more than 2 inputs.

Levers & Switches

Levers and switches are your signal sources. Connect their output wire directly into any gate input. That's it — no extra setup needed.

1 Lever ON — active signal
0 Lever OFF — no signal

Delay Block

Passes a signal through after a fixed wait. Set anywhere between 0.05s and 10s. Useful for timing sequences, preventing rapid toggling, or building a simple clock pulse.

3+ Input Gates

Every gate in BaBFT takes exactly 2 inputs. To AND three signals together, chain two AND gates — first gate handles A & B, second gate handles that result & C.

Inverting a Signal (NOT)

There is no NOT gate. Use a NOR gate with one input permanently wired to 0 — in-game, leave that input disconnected or tie it to a lever that is always off.

Half Adder

Your first real circuit — adds two 1-bit numbers using one XOR and one AND gate.

A half adder takes two inputs and produces two outputs. Connect two levers, wire both into a XOR gate and an AND gate in parallel.

  • Sum = XOR(A, B) — is 1 when exactly one input is 1
  • Carry = AND(A, B) — is 1 only when both are 1, meaning the result overflowed into the next bit

2-to-4 Decoder

2 input bits select exactly 1 of 4 output lines. The building block of any address-based circuit.

A decoder reads a binary number and turns on one specific output. With 2 bits you have 4 possible values (00, 01, 10, 11) so you get 4 outputs — only one is ever active at a time.

Each output is an AND gate wired to the right mix of A, B, and their inverses. The inverses come from NOR(x, 0) — BaBFT's only way to invert.

Parts needed

2 Levers — inputs A and B
2 NOR gates — one for NOT A, one for NOT B
4 AND gates — one per output line

How to wire it

  1. Place lever A and lever B. These are your address inputs.
  2. Place a NOR gate. Connect lever A → input 1. Leave input 2 disconnected (disconnected = 0 in BaBFT). The output is NOT A.
  3. Place another NOR gate. Connect lever B → input 1. Leave input 2 disconnected. Output is NOT B.
  4. Place an AND gate for Out 0. Wire NOT A → input 1, NOT B → input 2. Active only when both levers are off.
  5. Place an AND gate for Out 1. Wire NOT A → input 1, lever B → input 2. Active only when A=off, B=on.
  6. Place an AND gate for Out 2. Wire lever A → input 1, NOT B → input 2. Active only when A=on, B=off.
  7. Place an AND gate for Out 3. Wire lever A → input 1, lever B → input 2. Active only when both levers are on.

Scaling up in BaBFT

A full 2-to-4 decoder costs 2 NOR gates (inverters) + 4 AND gates = 6 gates total. To address 30 pixels you need a 5-bit decoder — 5 NOR gates for the inverters, then up to 32 AND chains (one per address), each AND chain being 4 two-input AND gates deep. That's around 130 gates just for the address logic.
